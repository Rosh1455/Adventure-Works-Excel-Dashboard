# Adventure Works Sales Dashboard (Excel)

## Overview

An interactive sales dashboard built in Microsoft Excel to analyze sales performance, customer trends, regional sales, and key business KPIs.

## Features

- Interactive Dashboard
- KPI Cards
- Pivot Tables
- Pivot Charts
- Slicers
- Sales Trend Analysis
- Region-wise Sales Analysis
- Product-wise Sales Analysis
- Top Customer Profit Analysis

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Data Validation
- Formulas
- Dashboard Design

## Business Insights

- Identified top-performing products.
- Compared sales across different regions.
- Tracked monthly sales trends.
- Analyzed customer profitability.
- Visualized KPIs for business decision-making.

- Data Analysis
- Dashboard Development
